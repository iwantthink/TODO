# 模拟器无法抓包

[macbook 上安卓模拟器无法连接网络问题记录](https://blog.csdn.net/qq_28268507/article/details/83659299)


1. 右上角点击wifi — 打开网络偏好设置

2. 底部高级 — 选择DNS

3. 添加DNS 服务器 8.8.8.8

4. 保存 — 应用

5. 重启模拟器即可
